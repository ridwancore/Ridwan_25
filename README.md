# Phishing URL Detector

This project uses Machine Learning to analyze and identify malicious URLs to prevent phishing attacks.

## 📁 Project Structure
Based on the files in this repository:

* **app.py**: The main application entry point.
* **train_model.py**: Script used to train the machine learning model.
* **predict.py**: Handles the prediction logic for new URLs.
* **feature_extraction.py**: Extracts specific characteristics from URLs for analysis.
* **phishing_model.pkl**: The saved trained model file.
* **malicious_phish.csv**: The dataset used for training (Note: This is a large 44MB file).
